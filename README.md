# sled-vl

## 简介  
`sled-vl` 是专为 [sled] 数据库设计的可视化工具，旨在为开发者提供直观界面，用于浏览数据库结构、查询键值数据、分析数据关系，降低使用sled数据库的门槛。

## 状态  
🚧 **开发中** 🚧  

⚠️ 重要提示：本项目目前处于早期开发阶段，功能尚未稳定，数据处理可能存在边缘场景漏洞，且开发过程中可能引入不兼容更新。**请务必谨慎使用**，切勿用于生产环境或存储关键数据。

## 计划核心功能  
- 连接并浏览sled数据库实例  
- 键值对可视化展示（树形/列表视图）  
- 基础查询与过滤功能  
- 数据导出（JSON/CSV格式）  
- 数据结构关系可视化  
- 轻量级性能监控
